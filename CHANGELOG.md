# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-03-01

### Added
- SearXNG one-click deployment in install.sh
- `--with-searxng` / `-s` flag for automatic Docker installation
- Detailed SearXNG setup instructions in README
- Verification commands for SearXNG
- VISION.md - Project vision and direction
- CONTRIBUTING.md - Contribution guidelines
- SECURITY.md - Security policy and best practices

### Updated
- README.md - More concise, OpenClaw-style
- README_ZH.md - Matching English version

### Added
- Initial release
- Smart source selection (auto-detect Chinese/English/code)
- GLM webSearchPrime integration
- SearXNG integration
- GitHub API integration
- Result caching (1-hour TTL)
- Automatic deduplication
- Time-based filtering
- Parallel search (all sources)
- Unified result format
- Bilingual documentation (EN + ZH)
- One-click install script
