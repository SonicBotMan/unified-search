# Configuration References

This folder contains sample configuration files.

## mcporter-sample.json

Copy this file to your mcporter config location and fill in your API keys:

```bash
# For OpenClaw users:
cp mcporter-sample.json ~/.openclaw/workspace/config/mcporter.json

# Edit with your API keys
nano ~/.openclaw/workspace/config/mcporter.json
```
