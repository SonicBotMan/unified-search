## How to Contribute

1. **Bug fixes** → Open a PR!
2. **New features** → Open an issue first to discuss
3. **Questions** → GitHub Issues

## Pull Request Guidelines

- One PR = one issue/topic
- Keep changes focused and small
- Include tests if applicable
- Update documentation for any changes

## Development Setup

```bash
# Clone
git clone https://github.com/SonicBotMan/unified-search.git
cd unified-search

# Install dependencies
pip install -r requirements.txt

# Test
python unified-search.py "test query"
```

## Code Style

- Python: Follow PEP 8
- Use meaningful variable names
- Add comments for complex logic
