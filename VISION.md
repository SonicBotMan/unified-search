# Unified Search Vision

Unified Search is an intelligent search tool that combines multiple search engines into a single, powerful interface.

## Goal

A search tool that is:
- **Easy to use** - One command to search everything
- **Smart** - Automatically selects the best search engine
- **Private** - Can run locally with SearXNG
- **Extensible** - Easy to add new search sources

## Current Focus

### Priority
- ✅ Multiple search engine integration
- ✅ Smart source selection
- ✅ Local deployment (SearXNG)
- 🔄 Caching and deduplication
- 🔄 Parallel search

### Next Priorities
- More search source integrations (exa, Brave, etc.)
- Web UI for configuration
- OpenClaw skill integration
- Result ranking algorithm

## What We Will Not Merge

- Paid service integrations without clear open-source alternative
- Features that require proprietary APIs
- Large UI frameworks (keep it CLI-focused)
